# mochi

- 스프링부트
- JPA
- mysql
- jsp
- vue.js

### 주요 기능 범위
- 헤더, 푸터, LNB
- 회원관리(세션)
- 디바이스 구분 처리 필터(웹페이지)
- 반응형페이지
- 게시판(페이징처리)
- 댓글, 대댓글
